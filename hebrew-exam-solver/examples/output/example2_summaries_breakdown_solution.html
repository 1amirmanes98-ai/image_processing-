תמציות מרכז ופיזור, נקודות שבירה, ציוני תקן וקו הרגרסיה — פתרון מלא

פתרון מפורט עם הסבר לכל מעבר + תיבות על הטריקים והרעיונות החשובים
ההגדרות שנעבוד איתן (אלו ההגדרות הסטנדרטיות בקורס):
תמצית למרכז: ‏T: ℝⁿ → ℝ כך שלכל a > 0, b ∈ ℝ מתקיימת שקילות למיקום ולסקלה: T(aX + b·1) = a·T(X) + b.
תמצית לפיזור: ‏T(X) ≥ 0, אדישה להזזה וכופלת ב-|a|: לכל a, b: ‏T(aX + b·1) = |a|·T(X).
נקודת שבירה: ‏ε*ₙ = m*/n, כאשר m* הוא המספר הקטן ביותר של תצפיות שיריב יכול להחליף בערכים שרירותיים כך שהתמצית "תתפוצץ" (תשאף ל-±∞). נקודת שבירה אסימפטוטית: הגבול כאשר n → ∞. תזכורת לגבי שברונים: לפי ההגדרה x_p = inf{x : F̂(x) ≥ p}, השברון האמפירי הוא סטטיסטי הסדר x_p = X₍ₖ₎ עם k = ⌈np⌉.

שאלה 2 — האם התמציות הבאות חוקיות? ומהן נקודות השבירה?

2.1 המקסימום T(X) = X₍ₙ₎ — תמצית למרכז? כן

בדיקת ההגדרה: עבור a > 0 הפונקציה x ↦ ax+b עולה ממש, ולכן היא שומרת על הסדר בין התצפיות — האיבר הגדול ביותר נשאר הגדול ביותר:

T(aX+b) = max_i (aXᵢ + b) = a·max_i(Xᵢ) + b = a·T(X) + b ✓

כלומר המקסימום מקיים את ההגדרה הפורמלית — הוא תמצית למרכז חוקית (וגם מקיים X₍₁₎ ≤ T ≤ X₍ₙ₎ באופן טריוויאלי). זה לא אומר שהוא תמצית טובה.

נקודת שבירה: מספיק לקלקל תצפית אחת — מזיזים אותה ל-+∞ והמקסימום שואף ל-∞. לכן ε*ₙ = 1/n, ואסימפטוטית: 1/n → 0.
הטריק/הלקח (וזו כל הפואנטה של התרגיל): ההגדרה של תמצית למרכז תופסת רק שקילות לטרנספורמציה לינארית עולה — היא לא תופסת "אמצעיות". לכן דברים מוזרים כמו המקסימום עוברים אותה. ההבחנה בין תמציות מגיעה מנקודת השבירה: מקסימום חוקי אבל שביר לחלוטין (0), לעומת חציון עם נקודת שבירה ½ — הרובסטית ביותר האפשרית. המעבר המרכזי בהוכחות מהסוג הזה: פונקציה עולה שומרת על סטטיסטי הסדר, כלומר (aX+b)₍ᵢ₎ = aX₍ᵢ₎+b עבור a>0.

2.2 ממוצע משוקלל T(X) = Σwᵢ Xᵢ / Σwᵢ, ‏wᵢ ≥ 0 — תמצית למרכז? כן

בדיקת ההגדרה (בהנחה Σwᵢ > 0, אחרת לא מוגדר) — לינאריות פשוטה של הסכום:

T(aX+b) = Σwᵢ(aXᵢ+b)/Σwᵢ = (a·ΣwᵢXᵢ + b·Σwᵢ)/Σwᵢ = a·T(X) + b ✓

בנוסף, זהו צירוף קמור של התצפיות (מקדמים אי-שליליים שסכומם 1), ולכן תמיד X₍₁₎ ≤ T(X) ≤ X₍ₙ₎ — עוד חיזוק לכך שזו תמצית מרכז סבירה. הממוצע הרגיל הוא המקרה הפרטי wᵢ = 1.

נקודת שבירה: קיים לפחות i אחד עם wᵢ > 0. היריב מחליף דווקא את התצפית הזו ב-M → ∞, והתמצית מכילה את האיבר wᵢM/Σwⱼ → ∞. לכן ε*ₙ = 1/n → 0 — בדיוק כמו הממוצע הרגיל.
חשוב ללמוד: שקלול לא קונה רובסטיות: כל עוד למישהו יש משקל חיובי, קלקול שלו מפוצץ את התמצית. נקודת השבירה מוגדרת לפי המקרה הגרוע ביותר (היריב בוחר את מי לקלקל) — טעות נפוצה היא לחשוב שאם יש תצפיות עם משקל 0 אז התמצית מוגנת.

2.3 הטווח T(X) = X₍ₙ₎ − X₍₁₎ — תמצית לפיזור? כן

אי-שליליות: תמיד X₍ₙ₎ ≥ X₍₁₎, ולכן T ≥ 0 ✓.

שקילות לסקלה: נבדוק את שני המקרים. עבור a > 0 הסדר נשמר:

T(aX+b) = (aX₍ₙ₎+b) − (aX₍₁₎+b) = a(X₍ₙ₎−X₍₁₎) = |a|·T(X) ✓

עבור a < 0 הסדר מתהפך: המקסימום החדש מגיע מהמינימום הישן ולהפך — max(aX+b) = aX₍₁₎+b ו-min(aX+b) = aX₍ₙ₎+b:

T(aX+b) = (aX₍₁₎+b) − (aX₍ₙ₎+b) = a(X₍₁₎−X₍ₙ₎) = (−a)(X₍ₙ₎−X₍₁₎) = |a|·T(X) ✓
נקודת שבירה: קלקול תצפית אחת ל-+∞ שולח את המקסימום, ולכן את הטווח, ל-∞: ‏ε*ₙ = 1/n → 0.
טריק נפוץ בהוכחות פיזור: תמיד לפצל ל-a>0 ו-a<0, ולזכור שכפל במספר שלילי הופך את הסדר: ‏(aX)₍ᵢ₎ = aX₍ₙ₊₁₋ᵢ₎. שכחת המקרה a<0 היא הורדת הנקודות הקלאסית בשאלות האלה. וגם: ההזזה b תמיד מתבטלת בהפרש — לכן תמציות פיזור בנויות מהפרשים.

2.4 העשירון התחתון T(X) = X₀.₁ — תמצית למרכז? כן

בדיקת ההגדרה: ‏X₀.₁ = X₍ₖ₎ עם k = ⌈0.1n⌉ — סטטיסטי סדר. עבור a > 0 הסדר נשמר, ולכן:

T(aX+b) = (aX+b)₍ₖ₎ = aX₍ₖ₎ + b = a·T(X) + b ✓

וגם X₍₁₎ ≤ X₍ₖ₎ ≤ X₍ₙ₎. כלומר, שוב — לפי ההגדרה הפורמלית זו תמצית למרכז חוקית, למרות שאינטואיטיבית העשירון התחתון בכלל לא "באמצע".

נקודת שבירה: כדי לשלוח את X₍ₖ₎ ל-−∞ צריך שלפחות k תצפיות יהיו ב-−∞, ומספיק לקלקל בדיוק k. (הכיוון השני, ל-+∞, יקר יותר: דרושות n−k+1 ≈ 0.9n תצפיות.) לכן:
ε*ₙ = ⌈0.1n⌉/n   ⟶   0.1 אסימפטוטית
חשוב ללמוד (הכלל הכללי): לשברון ה-p יש נקודת שבירה אסימפטוטית min(p, 1−p) — הצד הזול לקלקול קובע. מציבים p=0.5 ומקבלים את התוצאה המפורסמת: החציון עם נקודת שבירה ½, המקסימלית האפשרית. p=0.1 נותן 0.1. נוסחה ששווה לשנן — היא עונה על משפחה שלמה של שאלות.

2.5 T(X) = X₀.₉ − X₀.₁ (הטווח הבין-עשירוני) — תמצית לפיזור? כן

אי-שליליות: ‏X₀.₉ = X₍⌈0.9n⌉₎ ≥ X₍⌈0.1n⌉₎ = X₀.₁ כי ⌈0.9n⌉ ≥ ⌈0.1n⌉, ולכן T ≥ 0 ✓.

הזזה: שני השברונים זזים ב-b וההפרש מבטל אותה ✓. סקלה: עבור a>0 שני השברונים נכפלים ב-a ⟸ ‏T(aX)=aT(X). עבור a<0 הסדר מתהפך והשברונים מתחלפים בתפקידים — העשירון העליון של הנתונים החדשים מגיע מהעשירון התחתון של הישנים:

(aX)₀.₉ = a·X₀.₁ ,   (aX)₀.₁ = a·X₀.₉  ⟹  T(aX) = a·X₀.₁ − a·X₀.₉ = (−a)(X₀.₉−X₀.₁) = |a|·T(X) ✓

(עד כדי דקויות של ההגדרה עם inf בקצוות — ברמת הקורס זו התשובה.)

נקודת שבירה: הדרך הזולה ביותר לפוצץ: לשלוח את X₀.₁ ל-−∞ (עולה ⌈0.1n⌉ קלקולים) או את X₀.₉ ל-+∞ (עולה n−⌈0.9n⌉+1 ≈ 0.1n). בשני המקרים כ-10% מהתצפיות:
ε*ₙ ≈ 0.1 + O(1/n)   ⟶   0.1 אסימפטוטית
השוואה ששווה לזכור: הטווח הרגיל (2.3) והטווח הבין-עשירוני מודדים "אותו דבר", אבל הראשון נשבר עם תצפית אחת (0) והשני דורש 10% מהנתונים (0.1). קיצוץ הזנבות הוא בדיוק המנגנון שקונה רובסטיות — אותו רעיון כמו IQR (עם 0.25) וממוצע קטום. שאלת השוואה כזו היא חומר קלאסי למבחן.

2.6 מרכז הטווח פחות החציון — תמצית לפיזור? לא

T(X) = (X₍₁₎+X₍ₙ₎)/2 − median(X)

התמצית אמנם אדישה להזזה (שני האיברים שקולים למיקום וההפרש מבטל את b), אבל היא נכשלת בשתי דרישות:

1) אי-שליליות נכשלת — מספיקה דוגמה נגדית אחת: ניקח X = (0, 9, 10). אז מרכז הטווח = (0+10)/2 = 5, החציון = 9, ולכן:

T(X) = 5 − 9 = −4 < 0 ✗

2) גם שקילות הסקלה נכשלת: עבור a < 0 שני הרכיבים שקולים למיקום, ולכן T(aX) = a·T(X). כאשר T(X) ≠ 0 זה שווה −|a|T(X) ≠ |a|T(X) ✗.

נקודת שבירה (לשם שלמות): גם אילו הייתה חוקית — קלקול תצפית אחת מפוצץ את מרכז הטווח, כלומר 1/n → 0.
מה ללמוד מכאן: הפרש בין שתי תמציות מרכז הוא לא תמצית פיזור — הוא מודד א-סימטריה (צידוד): חיובי כשהזנב הימני ארוך, שלילי כשהשמאלי ארוך. והטריק הטכני החשוב: כדי להפריך, לא מוכיחים — בונים דוגמה נגדית מספרית קטנה (3 תצפיות מספיקות). לחפש דוגמה נגדית לפני שמנסים להוכיח זה הרגל ששווה זהב במבחני נכון/לא נכון.

שאלה 1 — לציוני תקן ממוצע 0 וסטיית תקן 1

ציון התקן: zᵢ = (xᵢ − x̄)/s, כאשר s סטיית התקן המדגמית (ומניחים s > 0).

ממוצע: מלינאריות הסכום — הקבועים יוצאים החוצה, וסכום הסטיות מהממוצע הוא אפס:

z̄ = (1/n)·Σᵢ (xᵢ−x̄)/s = (1/s)·[(1/n)Σxᵢ − x̄] = (1/s)·(x̄ − x̄) = 0

סטיית תקן: מציבים z̄ = 0 בהגדרת השונות המדגמית, ומזהים בפנים בדיוק את השונות של x:

s²_z = (1/(n−1))·Σᵢ(zᵢ − z̄)² = (1/(n−1))·Σᵢ zᵢ² = (1/s²)·(1/(n−1))·Σᵢ(xᵢ−x̄)² = s²/s² = 1

ולכן s_z = 1. ∎ (דרך שקולה: ‏z = (1/s)·x − x̄/s היא טרנספורמציה לינארית, ולפי הזהויות מהדף: הממוצע עובר לינארית והשונות נכפלת ב-(1/s)² .)

חשוב ללמוד: תקנון הוא בדיוק "להשתמש בכך שממוצע הוא תמצית למרכז וסטיית תקן תמצית לפיזור": מחסרים את המרכז ומחלקים בפיזור ⟸ מקבלים גודל חסר יחידות עם מרכז 0 ופיזור 1. שני המעברים שחוזרים בכל הוכחה כזו: ‏Σ(xᵢ−x̄) = 0 ולינאריות. זה גם הבסיס לסעיף הבא ולמשפט הגבול המרכזי.

שאלה 2 — קו הרגרסיה על ציוני תקן הוא z_Y = r·z_X

מריצים רגרסיה פשוטה של z_Y על z_X ומשתמשים בנוסחאות הריבועים הפחותים שפותחו בדף (עבור נתונים כלשהם: שיפוע b = s_{XY}/s²_X וחותך a = Ȳ − bX̄), כשמציבים את נתוני ה-z:

שלב 1 — השיפוע. המונה הוא השונות המשותפת של ציוני התקן. מכיוון ש-z̄_X = z̄_Y = 0 (שאלה 1), היא בדיוק ההגדרה של מקדם המתאם של פירסון (כפי שמופיע בדף — r הוא "הממוצע המתוקן של מכפלות ציוני התקן"):

s_{z_X z_Y} = (1/(n−1))·Σᵢ(z_{Xᵢ}−z̄_X)(z_{Yᵢ}−z̄_Y) = (1/(n−1))·Σᵢ ((Xᵢ−X̄)/s_X)·((Yᵢ−Ȳ)/s_Y) = r

המכנה הוא שונות ציוני התקן של X, ששווה 1 (שאלה 1). לכן:

b = s_{z_X z_Y} / s²_{z_X} = r / 1 = r

שלב 2 — החותך. קו הריבועים הפחותים עובר תמיד דרך נקודת הממוצעים (z̄_X, z̄_Y) = (0,0):

a = z̄_Y − b·z̄_X = 0 − r·0 = 0
ולכן קו הרגרסיה:  ẑ_Y = r·z_X  ∎
למה זו תוצאה חשובה (ולא רק תרגיל): (1) היא נותנת את הפירוש האמיתי של r: סטייה של סטיית תקן אחת ב-X מנבאת סטייה של r סטיות תקן ב-Y. (2) מכיוון ש-|r| ≤ 1, הניבוי תמיד קרוב יותר לממוצע מהתצפית — זו בדיוק "הנסיגה אל הממוצע" (regression to the mean), ומכאן השם רגרסיה! (3) חוזרים ליחידות המקוריות ומקבלים מיד את השיפוע הכללי: ‏b = r·s_Y/s_X. שלישיית עובדות שחוזרת במבחנים שוב ושוב.

סיכום — הטריקים שחוזרים

  • a>0 שומר סדר, a<0 הופך סדר: ‏(aX+b)₍ᵢ₎ = aX₍ᵢ₎+b או aX₍ₙ₊₁₋ᵢ₎+b בהתאמה — המעבר המרכזי בכל הוכחות השקילות.
  • ההגדרות פורמליות, לא אינטואיטיביות: מקסימום ועשירון עוברים את הגדרת ה"מרכז". מה שמבדיל תמצית טובה מרעה זו נקודת השבירה.
  • נקודת שבירה של שברון p: ‏min(p, 1−p); חציון = ½ (המקסימום האפשרי); ממוצע/מקסימום/טווח = 0.
  • הפרכה = דוגמה נגדית קטנה, לא הוכחה. 3 תצפיות בדרך כלל מספיקות.
  • ציוני תקן: ‏Σ(xᵢ−x̄)=0 + לינאריות ⟸ ממוצע 0, ס״ת 1; ומהם ẑ_Y = r·z_X ⟸ הנסיגה אל הממוצע והשיפוע r·s_Y/s_X.
